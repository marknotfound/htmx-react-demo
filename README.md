# Running
```
# Clone the repo
git clone git@github.com:marknotfound/htmx-react-demo.git
cd htmx-react-demo

# Create + activate a virtual environment
python3 -m venv venv
source venv/bin/activate

# Install dependencies (flask)
pip install -r requirements.txt

# Run
flask run
```
